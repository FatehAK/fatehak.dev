My curated collection of resources I've explored so far are listed here. I've left out some commonly known resources and only added ones which I felt were a little unknown.

This list is live and will be kept updated as I find more awesome resources.

## Tooling

- [progressive-tooling](https://progressivetooling.com/) - links to various open source tools, a goldmine of a find really!
- [grep.app](https://grep.app/) - my personal fav, it makes seaching across github repos blazingly fast
- [IT Tools](https://it-tools.tech/) - an all in one tool collection
- [WebpageTest](https://www.webpagetest.org/) - industry recognized tool to visualize your web page's request timeline
- [Symbols](https://symbols.wentin.net/) - one stop shop for copying unicode symbols
- [jsnice](http://www.jsnice.org/) - js deobfuscation tool, super handy for the reverse engineers out there
- [sli.dev](https://sli.dev/) - create great looking presentation slides using code
- [Security Headers](https://securityheaders.com/) - validate your website's security headers
- [crontab](https://crontab.guru/) - easily create `cron` job schedule expressions
- [browserling](https://www.browserling.com/) - free cross browser testing tool
- [badssl](https://badssl.com/) - visualize issues due to improper SSL configuration
- [gimme-an-img](https://just-gimme-an-img.vercel.app/) - create an optimized `<img />` tag
- [useful-forks](https://useful-forks.github.io/) - find useful forks of a github repository

## UI Components

Non Framework:

- [ui-verse](https://uiverse.io/all) - tons of components to choose from
- [ui-buttons](https://ui-buttons.web.app/) - that's a lot of buttons!
- [toggles.dev](https://toggles.dev/) - website dedicated to theme toggles, why not!
- [shoelace](https://shoelace.style/) - web components for different UI primitives
- [free-frontend](https://freefrontend.com/) - mix of different UI components
- [Daisy UI](https://daisyui.com/) - tailwind based

Framework oriented:

- [Radix UI](https://www.radix-ui.com/) - they come unstyled, perfect as a good base
- [Mantine](https://mantine.dev/) - react based
- [Headless UI](https://headlessui.com/) - react + tailwind based

## CSS

- [clay.css](https://codeadrian.github.io/clay.css/) - a cool 3D-isque shadow effect for your cards
- [smolcss.dev](https://smolcss.dev/) - tiny snippets to achieve various layouts
- [CVA](https://cva.style/docs/) - class based component approach
- [open-props](https://open-props.style/) - css variable first utilites
- [primer](https://primer.style/) - github's official design system
- [picocss](https://picocss.com/) - minimal css base
- [milligram](https://milligram.io/) - another minimal css base, they keep on coming!
- [half-moon](https://www.gethalfmoon.com/) - dark theme supported css variables
- [shadow-palette](https://www.joshwcomeau.com/shadow-palette/) - create multi-layered shadows for added realism
- [gradient-generator](https://www.joshwcomeau.com/gradient-generator/) - create beautiful lush gradients
- [animista](https://animista.net/) - pure css keyframe animations

## JS Ecosystem

- [fresh](https://fresh.deno.dev/) - web framework built with deno
- [fastify](https://www.fastify.io/) - an alternative to Node's Express.js
- [trpc](https://trpc.io/) - offers end to end typesafe API's, must check this out for a fresh project!
- [neutralino.js](https://neutralino.js.org/) - a light weight alternative to Electron.js
- [mitosis](https://github.com/builderio/mitosis) - write components once, compile to every framework
- [neo](https://github.com/neomjs/neo) - a web worker driven rendering framework, most interesting!
- [quicklink](https://github.com/GoogleChromeLabs/quicklink) - prefetch resources in the background when idle
- [comlink](https://github.com/GoogleChromeLabs/comlink) - makes working with web workers a breeze!
- [memlab](https://facebook.github.io/memlab/) - profile and detect memory leaks in your code
- [swup](https://swup.js.org/) - seamlessly add page transition effects

React specific:

- [framer-motion](https://www.framer.com/motion/) - production-ready motion library for react
- [remotion](https://www.remotion.dev/) - programmatic `.mp4` video generation, this should blow your mind!
- [react-flow](https://reactflow.dev/) - customizable flow charts in react
- [auto-animate](https://auto-animate.formkit.com/) - automatic animations for supported elements with a single line of code
- [linaria](https://github.com/callstack/linaria) - zero runtime css-in-js library

## Icons

Sites offering both `svg` and `png` formats for free!

- [SVG Repo](https://www.svgrepo.com/) - great collection of svg icons
- [Free Icons](https://freeicons.io/) - has free icons and illustrations
- [Ming Cute](https://www.mingcute.com/) - unstyled icons
- [Feather](https://feathericons.com/) - more unstyled icons

Sites with a mix of free and premium:

- [Flat Icon](https://www.flaticon.com/) - png's are always free here
- [Icons 8](https://icons8.com/icons) - mixed
- [Icon Finder](https://www.iconfinder.com/) - mixed

## Fonts

- [google-font-helper](https://gwfh.mranftl.com/fonts/) - helper to self host your google fonts
- [font-bunny](https://fonts.bunny.net/) - collection of open source fonts
- [font-style-matcher](https://meowni.ca/font-style-matcher/) - helps minimize [FOUC](https://en.wikipedia.org/wiki/Flash_of_unstyled_content) by providing a matching fallback font

## Design

- [mesh-gradient](https://meshgradient.in/) - create beautiful gradient images
- [reasonable.colors](https://reasonable.work/colors/) - collection of accessible color palettes
- [undraw](https://undraw.co/) - free illustrations
- [manypixels](https://www.manypixels.co/gallery?ref=producthunt) - more free illustrations

## Social

- [shields.io](https://shields.io/) - generate github `shields` for use on a repositiory's `README.md` file
- [socialify.git](https://socialify.git.ci/) - create social banners for your github projects
- [socialsharepreview](https://socialsharepreview.com/) - visualize your `og` images for sharing on social media
- [emoji-picker](https://github-emoji-picker.vercel.app/) - copy an emoji's shortcode
- [meta-tags](https://webcode.tools/generators/meta-tags) - generate meta tags for SEO

## Self Hosted

- [Umami](https://umami.is/features) - self hosted analytics platform
- [Dashy](https://dashy.to/) - your very own personal dashbboard
- [Uptime Kuma](https://uptime.kuma.pet/) - nifty monitoring tool
- [Pikapods](https://www.pikapods.com/) - cloud offering for self hosted apps
- [ChangeDetection](https://github.com/dgtlmoon/changedetection.io) - monitor changes in a website (eg. it can monitor for price drops of a product and so on)

## AI

- [regexgo](https://www.regexgo.com/) - regex patterns powered by AI? count me in!
- [bigjpg](https://bigjpg.com/) - upscales your images, best free one I've come across!
- [calligrapher.ai](https://www.calligrapher.ai/) - converts text to beautiful handwriting

## Docs

- [frontend-checklist](https://frontendchecklist.io/) - checklist to follow for every website
- [frontend-perf-checklist](https://github.com/thedaviddias/Front-End-Performance-Checklist) - performance checklist for websites
- [browser-hacks](http://browserhacks.com/) - hacks to resolve cross-browser incompatibility issues
- [fugu-tracker](https://fugu-tracker.web.app/) - track development of web platform features
- [cssdb](https://cssdb.org/) - css spec tracker
- [glob-primer](https://github.com/isaacs/node-glob#glob-primer) - understand `glob` expressions easily
- [http-archive](https://httparchive.org/reports) - check this out to understand various metrics of the web
- [what-happens-when](https://github.com/alex/what-happens-when) - superb resource to learn what happens during certain events on the web
- [git-flight-rules](https://github.com/k88hudson/git-flight-rules) - my goto resource to resolve issues with `git`
- [you-dont-need-lodash](https://github.com/you-dont-need/You-Dont-Need-Lodash-Underscore) - native alternatives to lodash functions
- [project-guidelines](https://github.com/elsewhencode/project-guidelines) - guidelines to follow when creating a fresh web project
- [clean-code-js](https://github.com/ryanmcdermott/clean-code-javascript) - the javascript clean code reference
- [tailwind-cheat-sheet](https://nerdcave.com/tailwind-cheat-sheet) - handy cheatsheet for tailwind.css
- [jamstack.org](https://jamstack.org/generators/) - useful site listing various static site generators and CMS solutions available
